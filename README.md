## 🛠️ Installation & Setup

Clone the repository:

```
git clone https://github.com/ankitkr756208-hash/COMPLETE_REACT.git
```

Go to project directory:

```
cd COMPLETE_REACT
```

Install dependencies:

```
npm install
```

Run project:

```
npm run dev
```
